## 2.1 因果关系的反事实定义

### KKV 的核心定义

King, Keohane & Verba (1994) 在 *Designing Social Inquiry* 第三章中提出了一个基于反事实（counterfactual）的因果关系定义，试图为质性和量化研究提供统一的因果推断逻辑：

> **因果效应**（causal effect）= Y_i^T（单元 i 在接受处理时的结果）− Y_i^N（同一单元 i 在未接受处理时的结果）

这是一个关于**同一单元**在两种不同状态下的结果差异的定义。

### 因果推断的根本问题

从这个定义直接产生了一个根本性问题：

> **我们永远无法同时观察同一单元在接受处理和未接受处理两种条件下的结果。**

对于同一个国家、同一时间点，我们要么观察到它在民主制度下的经济表现，要么观察到它在威权制度下的表现——不可能同时观察到两者。因此，因果推断本质上是关于**反事实**的推断：我们必须推断"如果事情不是这样，会怎样？"

这意味着：**因果效应永远无法被确定性地知道，只能被估计。**

### 平均因果效应与个体效应

由于我们无法直接观察个体因果效应，KKV 转向估计**平均因果效应**（mean causal effect, β）：

- β = E[Y^T] − E[Y^N]（处理组与控制组的系统成分之差的期望值）
- 这需要跨多个单元的观察
- 需要将观察到的世界分解为**系统成分**（systematic component，规律的、持久的模式）和**非系统成分**（nonsystematic component，随机波动）

### 两个关键假设

要从观察数据中有效估计因果效应，需要满足两个假设：

**单元同质性（Unit Homogeneity）**
- 如果两个单元在关键解释变量上有相同的值，它们在因变量上的期望值也相同
- 更弱的版本——**常数效应**（constant effects）：因果效应的大小在所有单元间相同，即使单元的基准水平不同
- 这个假设是**不可检验的**（对于单个单元）；研究者必须指定假设适用的范围

**条件独立性（Conditional Independence）**
- 解释变量的值必须独立于因变量的值被赋予
- 随机分配（random assignment）自动满足这一条件；在观察研究中必须通过匹配或控制来近似满足
- 违反条件独立性 → 内生性（endogeneity）问题，解释变量本身被因变量所影响

### 理论构建五规则

KKV 提出了五条指导因果理论构建的规则：

1. **可证伪性**（Falsifiability）：理论必须指明什么证据可以推翻它；避免过多例外和特殊情况；主动寻找适用边界
2. **内部一致性**（Internal Consistency）：假设之间不能相互矛盾；形式化建模有助于揭示不一致
3. **选择因变量需谨慎**：因变量必须是"果"而非"因"（避免内生性）；不能选择使因变量恒定的观察
4. **最大化具体性**（Concreteness）：使用可观察而非不可观察的概念；避免同义反复（用结果作为原因的证据）
5. **尽可能广泛的适用范围**：在满足可证伪性和具体性的前提下，不要不必要地限制理论的适用范围

## 2.2 必要条件与充分条件因果观

### 质性传统的因果逻辑

与 KKV 的反事实/相关因果观不同，质性研究传统（特别是比较历史分析）使用一种基于数学逻辑的因果观：

**必要条件（Necessary Condition）**
- 定义：如果 X 不存在，Y 就不会发生。反事实表述：if not-X, then not-Y
- 集合论表述：Y 是 X 的子集（所有 Y=1 的案例都有 X=1）
- 例：国家崩溃是社会革命的必要条件（没有国家崩溃就不会有社会革命）

**充分条件（Sufficient Condition）**
- 定义：如果 X 存在，Y 就必然发生。覆盖律（covering law）表述：if X, then Y
- 集合论表述：X 是 Y 的子集（所有 X=1 的案例都有 Y=1）
- 例：特定的阶级联盟是民主化的充分条件

### INUS 因果（Mackie）

现实中的因果关系很少是简单的必要或充分条件。J.L. Mackie 提出了 **INUS 因果**概念来处理复杂因果关系：

> **INUS 条件**：一个**不充分但必要的**（Insufficient but Necessary）部分，属于一个**不必要但充分的**（Unnecessary but Sufficient）条件组合。

用布尔代数表达：

```
Y = (A · B · c) + (A · C · D · E)
```

其中：
- `·` 表示逻辑 AND（联合条件）
- `+` 表示逻辑 OR（替代路径）
- **小写字母表示变量取值为 0（不存在）**——这是布尔代数的标准约定，例如 `c` 表示变量 C 不存在/为假
- 括号内的每个联合项是一个充分条件（但不是必要的，因为有替代路径）
- 联合项内的每个因素是 INUS 条件

例如，在上面的公式中，B 是不充分的（单独存在不够），但在第一个联合项中是必要的，而第一个联合项本身是充分的但不必要的（因为还有第二条路径）。

### 与统计加性模型的对比

量化传统使用加性统计模型：

```
Y = β₀ + β₁X₁ + β₂X₂ + β₃X₃ + ε
```

Mahoney & Goertz (2006) 指出了两种模型之间的根本差异：

| 维度 | 布尔/集合论模型 | 统计加性模型 |
|------|---------------|------------|
| 基本运算 | 逻辑 AND、OR | 算术加法、乘法 |
| 因果的组合方式 | 联合充分条件 + 替代路径 | 独立效应的加权总和 |
| 变量关系 | 集合的超集/子集关系 | 线性回归中的参数估计 |
| 跨子集一致性 | 数学上保证（集合关系在任何子集中成立） | 不保证（参数估计可能因子样本而异） |
| 交互效应 | 内置于模型（AND 就是交互） | 需要额外添加交互项 |
| 单一变量的效应 | 取决于其他变量的组合（情境依赖） | 通常被估计为独立的固定效应 |
| 一个反面案例的影响 | 可能推翻必要性/充分性主张 | 通常被吸收到误差项中 |

**关键洞见**：布尔 AND **不是**乘法；布尔 OR **不是**加法。虽然统计学中有"交互项"来模拟 AND 效应，但这只是一种近似——质性传统中的联合因果（conjunctural causation）是一种根本不同的因果观念，不是统计交互效应的变体。

## 2.3 等终性（Equifinality）

### 定义

**等终性**（Equifinality）指的是：**同一结果可以由多条不同的因果路径产生**。

在布尔表达式中，等终性由 OR（`+`）运算符表示：

```
Y = (Path 1) + (Path 2) + (Path 3)
```

每条路径是一个不同的充分条件组合，但它们都产生相同的结果 Y。

### 在两种传统中的地位

| | 质性传统 | 量化传统 |
|---|---------|---------|
| 重要性 | 核心概念，经常被明确讨论 | 大多缺失或隐含 |
| 识别路径数量 | 少量、明确标识的路径 | 潜在无限多条路径，通常不明确标识 |
| 处理方式 | 通过类型学理论系统处理 | 通过控制变量和交互项间接处理 |

### 对 Mill 方法的挑战

等终性直接挑战了 Mill 方法（特别是求同法）的基本假设：

Mill 的求同法假设：如果多个案例有相同的结果，那么这些案例一定共享某个共同原因。

但如果等终性存在，不同案例可能通过完全不同的因果路径到达相同结果——**它们不需要共享任何共同原因**。求同法在这种情况下会产生系统性的假阳性（错误地将巧合的共同特征认定为原因）。

George & Bennett (Ch.8) 特别强调：等终性在社会科学中极为普遍——民主化可以通过精英协议、群众运动、外部干预等多条路径实现；经济发展可以通过出口导向、进口替代、资源出口等多条路径实现。忽视等终性是方法论上的严重失误。

### 对研究设计的含义

等终性意味着：
- 不能假设相似结果有共同原因
- 需要类型学理论来系统识别和区分不同的因果路径
- 需要过程追踪来验证特定案例走的是哪条路径
- 跨案例比较必须在"同一因果路径"内进行，而非简单地跨所有正面案例比较

## 2.4 因果机制

### 定义

George & Bennett (2005, Ch.1) 将**因果机制**（causal mechanisms）定义为：

> 在特定条件下，将自变量与因变量连接起来的独立、稳定的因素和过程。

因果机制不同于因果效应——因果效应回答"X 对 Y 的影响有多大？"，因果机制回答"X 通过什么过程影响 Y？"

**类比**：如果因果效应是"按下开关，灯就亮了"，那么因果机制就是"按下开关 → 电路闭合 → 电流流过灯丝 → 灯丝加热发光"——整个中间过程。

### 因果机制 vs. 因果效应的地位争论

这是 KKV 与 George & Bennett 之间最重要的分歧之一：

**KKV 的立场**：因果效应是首要的，因果机制在操作上重要但在逻辑上是次要的。
- 识别机制是有帮助的，但"因果效应"的定义才是因果推断的核心
- 因果机制的识别本身也需要因果推断（每一环都需要建立因果效应），因此有无穷回归的风险
- 机制的价值在于它为因果效应的估计提供了额外支持

**George & Bennett 的立场**：因果机制与因果效应同等重要。
- 仅知道 X 与 Y 的相关关系（即使是因果效应），如果不了解其中的机制，我们对因果关系的理解是不完整的
- 机制揭示了因果关系的"如何"和"为什么"，不仅是"有没有"
- 案例研究的核心优势正在于其识别机制的能力
- 科学实在论（scientific realism）作为哲学基础支持对机制的关注

### 因果过程观察 vs. 数据集观察

Henry Brady 和 David Collier (2004) 提出了一个有用的区分：

**因果过程观察（Causal-Process Observations, CPOs）**
- 关于特定情境和机制的洞见，可能是推翻或支持理论的"决定性证据"（smoking guns）
- 通常来自深度案例研究和过程追踪
- 单个 CPO 可以是非常强的证据（例如，发现一份揭示决策者动机的内部备忘录）

**数据集观察（Data-Set Observations, DSOs）**
- 数据矩阵中的标准化行——跨多个观察显示模式
- 通常来自大样本研究
- 单个 DSO 的信息量有限；力量来自数量和模式

这一区分说明了为什么质性和量化研究对"一个观察的重要性"有不同看法：在质性传统中，一个关键的因果过程观察可以推翻一个理论；在量化传统中，整体模式才是关键，单个观察只是噪音中的一个数据点。

## 要点总结

1. **因果效应的反事实定义是跨传统的**：无论质性还是量化研究，因果推断都涉及"如果情况不同，结果会如何"的反事实推理。差异在于如何实施这种推理。

2. **两种因果逻辑服务于不同目标**：质性传统追求"结果之因"（causes-of-effects），用集合论和INUS逻辑；量化传统追求"因之结果"（effects-of-causes），用统计和概率。

3. **等终性是社会科学的常态**：假设单一因果路径几乎总是错误的。研究设计必须为多条路径留出空间。

4. **因果机制的地位取决于你的认识论立场**：但无论立场如何，了解机制对于深化因果理解都是有价值的。
