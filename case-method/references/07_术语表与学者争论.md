| 共变法 | Method of Concomitant Variations | 检验变量的程度共变 | Mill [1843] |
| 一致性方法 | Congruence Method | 评估案例结果是否与理论预测一致 | George & Bennett 2005 |

## 7.2 学者间六大关键争论

### 争论一：推断逻辑的统一性

| | KKV (1994) | George & Bennett (2005) |
|---|-----------|----------------------|
| **立场** | 存在统一的推断逻辑，质性和量化研究应遵循相同的标准 | 两者共享认识论（都从理论推导、检验、修正），但方法论逻辑不同 |
| **核心论据** | 因果推断的反事实定义对所有研究都适用；增加观察数量是改善推断的通用策略 | 过程追踪基于案例内因果序列，不是跨案例协变分析的变体；"增加观察"不能改变理论本身 |
| **实践含义** | 案例研究应尽量模仿统计方法的逻辑，增加观察以减少不确定性 | 案例研究有独特的优势（概念效度、机制识别、复杂因果），不应被还原为统计方法的弱版本 |
| **核心张力** | 案例研究是否需要模仿统计方法来获得科学合法性？ |

### 争论二：因果机制 vs. 因果效应

| | KKV (1994) | George & Bennett (2005) |
|---|-----------|----------------------|
| **立场** | 因果效应是首要的；因果机制在操作上有用但逻辑上次要 | 因果机制与因果效应同等重要 |
| **核心论据** | 机制的识别本身也需要因果推断（每一环都需要估计因果效应），有无穷回归风险 | 仅知道效应而不知机制是不完整的理解；机制揭示"如何"和"为什么" |
| **实践含义** | 优先估计因果效应；机制作为辅助证据 | 投入与估计效应同等的精力来识别机制 |
| **核心张力** | 科学解释的目标是什么——知道"有多大影响"还是知道"如何影响"？ |

### 争论三：因变量选择

| | Geddes (1990) / KKV (1994) | Collier & Mahoney (1996) / G&B (2005) |
|---|---------------------------|--------------------------------------|
| **立场** | 基于DV选择原则上应被避免 | 在特定目的下合法且有价值 |
| **核心论据** | 违反条件独立性；导致系统性偏差（低估因果效应）；NICs案例展示了危害 | 取决于对照空间和研究目的；必要条件检验需要DV选择；因果异质性权衡使扩大样本不总是优选 |
| **允许例外** | 描述性推断；探索性假设生成（但不能用同一数据检验） | 以上加：Mill求同法；反事实推理；识别因果机制的启发式研究 |
| **核心张力** | 选择偏差的风险 vs. 因果异质性的风险——哪个更严重取决于具体研究情境 |

### 争论四：概念修改策略

| | Sartori (1970) | Collier & Mahon (1993) |
|---|---------------|----------------------|
| **立场** | 概念修改应通过经典阶梯：减少内涵以增加外延 | 不同范畴类型需要不同策略 |
| **核心论据** | 概念需要清晰边界和定义属性；经典分类是有效研究的基础 | 许多社会科学概念（如"民主"）不符合经典分类结构；家族相似和辐射范畴提供了更灵活的框架 |
| **实践含义** | 统一使用阶梯法修正概念 | 先判断范畴类型，再选择对应策略 |
| **核心张力** | 经典分类是否涵盖了所有重要的社会科学概念？ |

### 争论五：负面案例选择

| | 全案例设计倡导者 | Mahoney & Goertz (2004) |
|---|----------------|----------------------|
| **立场** | 应纳入所有可能的案例以避免选择偏差 | 应仅纳入结果有可能发生的案例（可能性原则） |
| **核心论据** | 任何排除都可能引入偏差；更多案例=更好的推断 | 纳入无关案例人为膨胀理论确认度；无关案例不提供真正的信息（乌鸦悖论） |
| **实例** | 冲突研究中纳入所有国家二元组 | 仅纳入政治相关二元组，排除不可能发生冲突的配对 |
| **核心张力** | 纳入不相关案例是否扭曲了推断？排除案例的标准是否客观？ |

### 争论六：质性与量化方法的地位

| | KKV (1994) | Mahoney & Goertz (2006) |
|---|-----------|----------------------|
| **立场** | 量化方法提供了推断的范式；质性方法应遵循相同标准 | 两种传统各有合法性；用一种传统的标准评判另一种是不当的 |
| **核心论据** | 一种推断逻辑、一种科学；质性方法可以在统一框架内改进 | 十个维度的根本差异源于不同的研究目标（causes-of-effects vs. effects-of-causes） |
| **实践含义** | 质性研究者应学习和应用统计推断原则 | 两种传统应理解彼此，但不应以对方的标准自我评判 |
| **核心张力** | 社会科学是一种科学还是两种？不同目标是否意味着不同的方法论标准？ |
