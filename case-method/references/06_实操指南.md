## 6.1 完整研究流程检查清单

### 第一步：明确研究谜题

- [ ] 你要解释什么结果（因变量）？
- [ ] 为什么这个结果令人困惑或有趣？（对照空间是什么？与什么比较使它成为需要解释的谜题？）
- [ ] 你追求的是"结果之因"（causes-of-effects，解释特定案例）还是"因之结果"（effects-of-causes，估计变量的一般效应）？

### 第二步：文献回顾与理论识别

- [ ] 现有理论对这个结果提出了哪些解释？
- [ ] 关键自变量有哪些？竞争理论之间的分歧在哪里？
- [ ] 是否有明确的理论可以检验？还是需要先生成假设？

### 第三步：概念构建

- [ ] 你的核心概念是否清晰界定？（Sartori：概念形成先于量化）
- [ ] 概念在抽象阶梯上的位置是否恰当？（HL/ML/LL哪个层次最适合你的比较范围？）
- [ ] 如果概念需要"旅行"到新案例，是否存在概念拉伸的风险？
- [ ] 你的概念属于哪种范畴类型？
  - 经典范畴 → 上移阶梯（减少属性）
  - 辐射范畴 → 添加修饰词（下移到子类）
  - 家族相似 → 识别交叉模式
- [ ] 外延与内涵是否符合反比变动律？

→ 详见：[01_概念构建与测量](01_概念构建与测量.md)

### 第四步：操作化与测量

- [ ] 是否区分了背景概念和系统化概念？（Adcock & Collier）
- [ ] 指标是否完整捕捉了系统化概念的内容？（内容效度）
- [ ] 是否考虑了聚合/区分效度？
- [ ] 跨情境比较是否需要情境特定指标或调整后的共同指标？
- [ ] 测量问题 vs. 概念问题是否被正确区分？

→ 详见：[01_概念构建与测量](01_概念构建与测量.md) 第1.3节

### 第五步：界定范围与选择案例

- [ ] 理论的范围条件是什么？（排除因果异质性区域）
- [ ] 是否应用了可能性原则？
  - 纳入规则：案例是否拥有至少一个相关自变量？
  - 排除规则：案例是否拥有消除性变量？
  - 无关案例是否已被排除？
- [ ] 根据研究目标选择了合适的设计？（见6.2决策矩阵）
- [ ] 是否评估了因果同质性 vs. 推广性的权衡？（Collier & Mahoney）
- [ ] 选择标准是否基于自变量而非因变量？（除非有充分理由）

→ 详见：[03_案例选择与选择偏差](03_案例选择与选择偏差.md)

### 第六步：选择分析方法

- [ ] 根据理论发展程度和案例数量选择方法（见6.3方法指南）
- [ ] 是否计划了一致性方法与过程追踪的组合？
- [ ] 如涉及多变量交互，是否考虑了类型学理论？

→ 详见：[04_研究方法_过程追踪与一致性](04_研究方法_过程追踪与一致性.md)、[05_类型学理论与研究设计](05_类型学理论与研究设计.md)

### 第七步：如使用类型学理论

- [ ] 是否构建了完整的属性空间？
- [ ] 是否按三个标准缩减了属性空间？（不可能类型、信息量丰富类型、匹配研究目标）
- [ ] 每种类型是否有对应的因果机制和预期结果？（区分类型学与类型学理论）
- [ ] 案例选择是否基于类型学空间？

→ 详见：[05_类型学理论与研究设计](05_类型学理论与研究设计.md)

### 第八步：进行案例内分析

- [ ] 是否使用了过程追踪来追踪因果机制的每一步？
- [ ] 因果链是否完整？是否有关键步骤的证据缺失？
- [ ] 是否主动考虑了替代因果假设？（避免确认偏差）
- [ ] 是否寻求了独特、具体的证据？（贝叶斯逻辑）

### 第九步：进行跨案例比较

- [ ] 如使用求同法/求异法，是否意识到其局限？（等终性、假设要求）
- [ ] 跨案例比较的结论是否被案例内分析所支持？
- [ ] 偏差案例是否被识别和分析？

### 第十步：处理等终性

- [ ] 是否考虑了多条因果路径？
- [ ] 不同案例是否可能通过不同机制达到相同结果？
- [ ] 如果存在等终性，是否通过类型学理论进行了系统处理？

### 第十一步：评估威胁

- [ ] 虚假相关：C和E是否都由Z导致？
- [ ] 因果优先性：C是否只是Z到E的中介？
- [ ] 因果深度：即使没有C，E是否仍会发生？
- [ ] 选择偏差：案例选择是否导致了系统性偏差？

### 第十二步：陈述结论

- [ ] 结论是否为条件性概括？（在什么条件下成立？）
- [ ] 范围条件是否明确？
- [ ] 选择限制是否被承认？
- [ ] 是否指出了需要后续研究的方向？

### 第十三步：反思研究贡献与局限

- [ ] 你的研究对该领域的理论贡献是什么？（发展、检验、修正、还是推翻了某个理论？）
- [ ] 你的案例研究设计的固有局限是什么？（推广性、因果识别的强度、证据可得性）
- [ ] 方法论选择中的权衡（如因果同质性 vs. 推广性）是否被坦诚讨论？
- [ ] 该研究为后续研究开启了什么具体问题？（而非泛泛的"需要更多研究"）

## 6.2 案例选择决策矩阵

| 研究目标 | 推荐设计 | DV选择 | 关键注意事项 |
|---------|---------|--------|------------|
| **假设生成** | 极端/异常案例 | ✓ 可接受 | 发现的模式需用新数据另行检验；警惕基于极端案例的过度推广 |
| **必要条件检验** | 全部正面案例 | ✓ 必需 | 检查所有正面案例是否都具有假设的必要条件；单个反例即可推翻 |
| **充分条件检验** | DV变异案例 | ✗ 不可 | 需要正面+相关负面案例；应用可能性原则选择负面案例 |
| **因果效应估计** | 基于IV选择 | ✗ 不可 | 需满足条件独立性；尽量在IV上有最大变异 |
| **理论发展** | 最相似+偏差案例 | 部分✓ | 先用DV选择发现模式，再扩展样本检验；结合案例内分析 |
| **理论检验** | 关键案例 | 取决于类型 | most-likely案例用于反驳；least-likely案例用于支持 |
| **偏差案例分析** | 偏离预期的案例 | N/A | 核心目的是精炼理论、发现遗漏变量 |
| **类型学分析** | 基于类型学空间 | 由类型决定 | 选择代表不同类型的案例；聚焦信息量丰富的类型 |

## 6.3 方法选择指南

| 条件 | 推荐方法 | 原因 |
|------|---------|------|
| 理论已充分发展，有明确预测 | 一致性方法 → 过程追踪确认 | 可以直接检验理论预测是否与案例结果一致，再用过程追踪验证因果链 |
| 理论不发达，变量不确定 | 启发式案例研究 + 过程追踪 | 先通过深入案例发现新变量和机制，再形成假设 |
| 少量案例（2-5个） | 最相似/最不同设计 + 过程追踪 | Mill方法提供跨案例比较框架，过程追踪提供案例内验证 |
| 中量案例（5-15个） | QCA或小N统计 | 布尔代数方法可识别等终性；新统计技术（bootstrap等）适用于小样本 |
| 复杂因果过程，多个中间步骤 | 过程追踪（分析性解释） | 追踪因果链的每一步，识别汇聚、交互和路径依赖 |
| 怀疑等终性存在 | 类型学理论 | 系统枚举所有变量组合，为每种组合指定不同的因果路径 |
| 单一案例 | 一致性（多理论竞争）+ 过程追踪 | 在一个案例内检验多个竞争理论的预测，用过程追踪区分 |
| 想要强力的理论检验 | 关键案例分析 | most-likely或least-likely案例提供相当于大N的检验力度 |
| 纵向变化的因果分析 | 前后比较设计 | 以关键转折点为界分为子案例，需在前后充分时间段观察 |

## 6.4 常见错误与防范

| 错误 | 说明 | 如何识别 | 防范措施 |
|------|------|---------|---------|
| **概念拉伸** | 扩大概念外延但不减少内涵，概念变得模糊 | 概念似乎可以适用于任何案例；不同案例中的含义不一致 | 使用Sartori阶梯；识别范畴类型；明确最小定义特征 |
| **DV选择偏差** | 仅研究有特定结果的案例推断因果 | 样本中因变量无变异或少变异；未检查负面案例 | 纳入负面案例；或明确限制推断范围（仅描述性/仅假设生成） |
| **混淆类型学与类型学理论** | 将案例分类但没有指定因果路径 | 只回答"是什么类型"而不回答"为什么" | 每种类型必须有对应的因果机制和条件性概括 |
| **忽视等终性** | 假设相同结果有共同原因 | 使用求同法时自动假设了唯一因果路径 | 主动考虑多条路径；使用类型学理论 |
| **过程追踪≠更多观察** | 将案例内证据当作统计意义上的独立观察 | 声称过程追踪"增加了N"而不解释其逻辑 | 理解其力量在于步骤间的逻辑连接，而非数量 |
| **一致性方法止于一致性** | 发现理论预测与结果一致就停止 | 没有检查虚假相关、因果优先性或因果深度 | 进行三种威胁评估；辅以过程追踪 |
| **纳入无关负面案例** | 将不可能出现结果的案例纳入分析 | 负面案例中大量缺乏任何相关自变量 | 应用可能性原则的纳入/排除规则 |
| **极端案例推广** | 将极端案例的特殊机制推广到一般案例 | 发现的复杂因果路径只在极端条件下存在 | 明确推断边界；用新样本检验 |
| **概念争议当测量问题** | 将关于概念"应该意味什么"的争论错误地当作测量效度问题 | 批评者实际上是在质疑系统化概念本身 | 区分Level 1争议（背景概念）和Level 2-4问题（测量效度） |
| **忽视因果异质性** | 盲目扩大样本以避免选择偏差，但混入了不同因果过程 | 跨不同区域的结论不一致；模型参数不稳定 | 评估因果同质性vs推广性的权衡 |

## 6.5 研究设计自查问题

在完成研究设计后，用以下问题自查：

**关于概念**：
1. 如果一个聪明的同行用完全不同的例子来说明你的核心概念，他们会选择什么？你会同意吗？
2. 你的概念能不能在不改变含义的情况下应用到你研究范围之外的案例？

**关于案例选择**：
3. 如果有人用不同的案例重做你的研究，结论会改变吗？如果会，这说明了什么？
4. 你的负面案例中，结果确实有可能发生吗？

**关于因果推断**：
5. 有什么证据能让你放弃你的假设？如果答不上来，你的理论可能不够具体。
6. 你能说出一个与你的假设同样一致但因果机制完全不同的替代解释吗？
7. 如果移除因果链中的某一步，你的论证是否崩溃？如果是，那一步的证据有多强？

**关于推广**：
8. 你的结论在什么条件下**不**成立？你清楚这些条件吗？
